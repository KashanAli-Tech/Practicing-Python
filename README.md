#  Practicing Python - Learning Systems Through Build

##  Overview

This repository documents my ongoing journey learning Python through hands on problem solving and system building. Rather than focusing only on isolated exercises, I aim to gradually build small, interactive systems that combine logic, structure, and user interaction.

The goal is to move from basic syntax understanding to structured programming to then small system design thinking.

---

##  What This Repo Contains

This repository includes a mix of:

###  Core Exercises
- Python fundamentals practice (loops, conditions, strings)
- Problem solving tasks (FizzBuzz variations, palindrome checks)

###  System-Based Projects
- Life Manager System (CLI based, multi tool system)
- Contacts manager (add/search/update/delete)
- Task manager (track completion state)
- Bank account system (OOP based with deposit/withdraw/view balance)
- Simple analytics tool (word counter)
- Grid Explorer (movement-based state system)

### Game Development Practice
- Tic Tac Toe (2-player terminal game)
- Win-condition detection (rows, columns, diagonals)
- Grid mapping system (1–9 input conversion)

---

##  Key Learning Focus

This repository is not about perfection in systems, instead it is about building understanding in:

- Procedural + functional programming  
- Basic object oriented programming (OOP)  
- State management in CLI systems  
- Control flow design (menu-driven applications)  
- Breaking problems into structured components  
- Iterative improvement through small builds
